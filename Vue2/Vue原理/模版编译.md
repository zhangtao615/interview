### 前置知识：with语法

```
const obj = {
  a: 100, 
  b: 200
}

// 使用with，等改变 {} 内自由变量的查找方式，将 {} 内自由变量，当作 obj 的属性查找
with(obj) {
  console.log(obj.a);
  console.log(obj.b);
  console.log(a) // 会报错
}
```

1. 会改变 {} 中自由变量的查找规则，当作obj属性来查找
2. 如果找不到匹配的obj属性，就会报错
3. with需要慎用，它打破了作用域规则，易读性变差
